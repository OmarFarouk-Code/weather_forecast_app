# weather_forecast_app
A real-time weather application built with Python, Tkinter, and OpenWeatherMap API.

#Features
Real-Time Data: Fetches current temperature, humidity, wind speed, pressure, and precipitation.
Dynamic UI: Features a responsive layout with visual feedback ("Fetching data...") during API calls.
Unit Standardization: Automatically converts and displays data in Metric units (Â°C, m/s).
Robust Error Handling: Validates user input and manages API errors (e.g., "City Not Found") or connection issues gracefully.
Enhanced UX: Includes "Enter" key binding for quick searching and auto-focus on the input field.

#Technical Requirements
Language: Python 3.x
Libraries: requests, python-dotenv, tkinter
API: OpenWeatherMap API Key

#Installation & Setup
Clone the repository
Install dependencies
Configure Environment Variables: Create a .env file in the root directory and add your API key
Run the application
